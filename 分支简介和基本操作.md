查看当前仓库的所有分支
```
git branch
```
新建分支
```
git branch <新分支> <版本号>
```
切换分支
```
git switch <分支>
```
新建并切换分支
```
git switch -c <分支> <版本号>
```
将其他分支合并到当前分支(**并产生一次提交**)
```
git merge <被合并分支>
```
变基(将本分支移动到其他的分支上)
```
git rebase <要移动到的分支>
```
删除已合并分支
```
git branch -d <被删除分支>
```
强制删除分支
```
git branch -D <被删除分支>
```
